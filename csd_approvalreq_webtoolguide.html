CSD T&C CONTRACTOR ENGINEER'S QUALIFICATION APPROVAL REQUEST TOOL

General Guidelines/Instructions

  • Except for the remarks and Location fields, all fields are mandatory to be filled up.
  • Engineer’s Name Should be as per the Name indicated in the IQAMA/Residence ID. It should be written as First Name Last Name.
  • (Important) Previous Approval if requesting for upgrade should be attached even if it was a downgrade revision.
  • 2 buttons are available at rightmost part of the form:
    • Generate PDF forms – Generates a zipped file of the approval forms in pdf of the engineers inputted.
      • Filename of the generated zipped file containing the PDF files is in this format:
        Company ID + "_prequalrequest" + "DateandTimenow(Format ddmmyyhhmm"
      • Filename of the PDF files are in this format:
        Company ID + "_CSDID_" + IqamaNo + "_RXX"
    • Generate Email Draft & PDF forms (in zipped file) - Generates an eml file which is a draft email format that commonly opens in ms outlook classic provided that there is already a Microsoft exchange account setup in the Computer where you will be accessing this tool.
      • The Same zipped file and PDF files will be generated as stated in 4.a.ii.
      • The eml file will have a filename in this format:
        Company ID + "_CSD_Approval_Draft_" + "DateandTimenow(Format ddmmyyhhmmnn"
  • (Important) If you have a Microsoft exchange account already setup in the computer where you will be using this tool, please open generate .eml file with outlook(classic).
    • It should contain already the generated PDF files as attachments and ready to be sent to CSD-Technical Support recipient to facilitate the interview request.
    • Just input the email addresses (to; cc, bcc as applicable).
    • If you want the eml file to be automatically opened by the outlook(classic) app, you can set it by default by:
      Going to windows settings > apps > default apps > search “set defaults for application” for outlook(classic) > click the .eml option > choose outlook(classic) > click “set as default”
  • If the .eml file fails to be generated, you can manually create any email draft and manually attach the generated pdf files of the approval requests.
  • (Important) THE GENERATED PDF FILES ARE NOT TO BE ALTERED, REVISED, AND EDITED ONCE GENERATED.
    If you have mistakenly input incorrect values, you must use the tool again to generate a revised version of the approval request form.
  • The Web tool is limited to inputting only 10 Engineer approval requests only:
    • The selected Region will be applicable to the engineers inputted in 1 session.
    • Please take note of this so the interview request will be as per the correct region.
