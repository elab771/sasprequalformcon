This page is a data source for the Engineers Qualification Tracker's Company Name dropdown.
It is not linked from the main form and does not need to be shared with anyone — it just needs to stay hosted in the same folder as the main form file, under the exact filename companieslist.html.

To add, edit, or remove a company: add, edit, or delete a <tr>…</tr> row inside the <tbody> below, keeping the same three columns in the same order (Company Name, Company ID, Voltage Level). Do not change the table's id="companiesTable" or the column order — the main form reads this table directly.
